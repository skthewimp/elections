# Legacy Archive

This folder contains the older election workspace that used to sit as a flat pile at the repository root.

The visible archive is now organized by function:

- `analyses/`: notebooks and rendered analysis outputs
- `scripts/`: standalone historical scripts
- `data/`: legacy datasets split by storage type
- `docs/`: reports and reference-style documents
- `media/`: saved plots and visual artifacts
- `spatial/`: shapefiles and related geometry assets
- `apps/`: preserved subprojects such as `figure`, `hebbal`, `parliament`, and `rounds`

`_flat_compat/` is not the canonical archive. It exists only so older filename-based scripts can still run against a flat namespace without forcing the human-facing archive back into that shape.
